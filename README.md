# Library Management System (C++)

This project is a simple Library Management System written in C++ using basic
Object-Oriented Programming concepts. It is intended for academic purposes
and beginner practice.

## Features
- Add and remove books
- Search books
- Register users
- Borrow and return books
- View books and users
- Menu-driven console interface

## How to Run
1. Compile the program:
   g++ library.cpp -o library
2. Run the program:
   ./library
   (or library.exe on Windows)

## Project Files
- `library.cpp` – Source code
- `Assignment_Library_Management_System.pdf` – Assignment documentation
